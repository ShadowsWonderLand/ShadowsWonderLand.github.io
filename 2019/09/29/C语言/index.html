C语言

发表于 2019-09-29 | 分类于 BLCU

一些看教材总结的小知识点

  1. 指数类型：e/E 之前必须有数字，且后面必须为整数
  2. 以0开头的是八进制 以0x开头的数是16进制
  3. 负数补码：该数的绝对值的二进制形式，按位取反再加1
  4. 在存放整数的存储单元中，最左边一位为0，该数为正数，为1为负数。
  5. Unsigned int：16位都用来存放数值本身 不包括符号
  6. 没有赋值的变量，其值是不确定的！
  7. 赋值语句是赋值表达式加一个句号，赋值表达式还可以用在其他语句中
  8. 条件运算符的结合方向是“自右至左”
  9. 运算>关系>条件运算符>赋值
  10. 传地址会改变实参值
  11. ‘/0’不算在有效字符串里面
  12. 进行字符数组输出时（printf”%S”）,输出的字符不包括“\0”,把空格当做分隔符
  13. 用gets（string）时，字符串结束标志“\0”也放在其中
  14. 用strcat（str1，str2）时，str2连接在str1后面，并把第一个后面的“\0”取消
  15. Strcpy（str1，str2）时，把str2复制到str1里面，同时拷贝“\0”，也可以用str n cpy(str1,str2,n)

把str2前n个复制到str1中

  1. strcmp(str1,str2) 如果str1<str2，值小于0
  2. Strlen（字符数组）测量的是有效长度
  3. Strlwr strupr
  4. 实参与形参的类型应相同或赋值兼容
  5. 值传递时改变形参的值不会改变实参值
  6. 函数的定义的第一行与函数的声明（函数原型）只差一个分号
  7. 数组名为实参时，对应的形参可以使数组名或指针变量
  8. 形参为数组名时可以不在方括号立面指明数组元素个数
  9. 数组作为形参时，传的是地址，所以形参变，实参变。因为数组名代表的是数组首元素的地址，要求形参为指针型变量，但因为C语言中下标法和指针法都可以访问一个数组，所以可以用数组名。
  10. Static int c:静态局部变量：在函数调用后不消失保持原值，属于静态储存，只赋以此初值，只能被本函数引用，未赋值是自动为0
  11. Extern：将外部变量的作用域扩展到其他文件 static 静态外部变量
  12. 指针常量不可改变值
  13. 数组可以在定义时整体赋初值 但不能在赋值语句中整体赋值
  14. 定义了指针后必须使它指向明确
  15. 指针变量的值可以改变 数组名代表地址常量 不能改变值
  16. 字符指针变量指向的内容是不可以被取代的
  17. .>* 故（*p）.num
  18. A-Z:65-90 a-z:97-122 0-9:48-57
